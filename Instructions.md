### UI Design Prompt

Design a clean, modern **AI generation input UI** similar to ChatGPT or image generation tools.

**Layout**

* A large rounded rectangular container centered horizontally.
* Light gradient background. Very subtle blue or gray.
* Inner container with white background, soft shadow, and a green outline border.
* Plenty of padding. Spacious and minimal.

**Main Input**

* A large single-line input field with placeholder text:
  **“What would you like to create?”**
* Placeholder text should be light gray and large enough to feel primary.
* Input should feel conversational and friendly.

**Left Side Controls**

* A small attachment or upload icon at the far left inside the input.
* Next to it, two pill-shaped dropdown buttons:

  * Button 1 label: **Style**
  * Button 2 label: **Aspect Ratio**
* Pills should have:

  * Light border
  * White background
  * Rounded corners
  * Dropdown arrow icon

**Right Side Action**

* A prominent green **Generate** button aligned to the right.
* Button should be pill-shaped.
* White text.
* Include a right-arrow icon inside the button.
* Button must visually stand out as the primary action.

**Visual Style**

* Minimal.
* Modern SaaS aesthetic.
* Smooth rounded corners everywhere.
* Soft shadows only. No harsh borders.
* Accessible color contrast.
* Desktop-first layout.

**Tech Constraints**

* Design should be implementable in **Next.js + Tailwind CSS (shadcn component)**.
* Components should be reusable.
* Responsive behavior required for tablet and mobile.
